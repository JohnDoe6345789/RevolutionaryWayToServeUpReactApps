# bootstrap/configs

This folder holds docs for the configuration helpers that populate provider fallbacks, logging directives, and loader settings consumed by the bootstrap runtime. Configs now live under `core`, `cdn`, `local`, and `helpers` subfolders in `bootstrap/configs`.

## Documents

- [`core/README.md`](core/README.md) – Core runtime configs for bootstrap, logging, module loading, and script manifests.
- [`cdn/README.md`](cdn/README.md) – CDN configs for dynamic modules, logging, network probing, and helper tools.
- [`local/README.md`](local/README.md) – Local loader configs for dependency resolution and compiler wiring.
- [`helpers/README.md`](helpers/README.md) – Helper configs for wiring local helper registries.
