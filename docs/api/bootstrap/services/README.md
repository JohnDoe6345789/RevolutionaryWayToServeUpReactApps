# bootstrap/services

Service helpers that power the bootstrap environment, including CDN helpers, local utilities, and shared registries used by the loader and renderers.

## Documents

- [`base-service.md`](base-service.md) – Describes the foundational service helpers every bootstrap service extends.
- [`service-registry-instance.md`](service-registry-instance.md) – Shows how the registry instance is populated with CDN/local services.
- [`service-registry.md`](service-registry.md) – Explains the API for retrieving services inside the loader/renderer.
