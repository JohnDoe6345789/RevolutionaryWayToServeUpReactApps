# Module: `bootstrap/services/cdn/network/provider-utils.js`

## Overview

- **Purpose:** Utility helpers for normalizing provider URLs and building alias maps.

## Globals
- `normalizeProviderBaseRawValue`
- `createAliasMap`
## Functions / Classes

- `normalizeProviderBaseRawValue` — ensures provider bases always end in a slash and use `https://` when needed.
- `createAliasMap` — converts alias objects into normalized maps consumed by the provider service.
