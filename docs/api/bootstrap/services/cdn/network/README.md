# bootstrap/services/cdn/network

Network-focused helpers that probe endpoints, normalize provider bases, and resolve module URLs for CDN loaders.

## Documents

- [`global-root-handler.md`](../../../constants/global-root-handler.md) – Describes how the shared global root and bootstrap namespace are detected.
- [`network-module-resolver.md`](network-module-resolver.md) – Walks through the resolver that probes provider bases for module URLs.
- [`network-probe-service.md`](network-probe-service.md) – Chronicles the probe service that loads scripts and retries CDN requests.
- [`network-provider-service.md`](network-provider-service.md) – Covers the provider service that tracks aliases, fallbacks, and proxy modes.
- [`provider-utils.md`](provider-utils.md) – Shows the helpers used to normalize provider bases and alias selections.
