# bootstrap/initializers/loaders

This folder holds docs for the loader helpers that produce the runtime `require`/`frameworkRender` helpers.

## Documents

- [`local-loader.md`](local-loader.md) – Combines Sass/TSX compilers, path utilities, and rendering helpers to build the bootstrap `require` function.
- [`local-module-loader.md`](local-module-loader.md) – Explains the loader that fetches local modules, caches compiled bundles, and exposes the helper namespace.
