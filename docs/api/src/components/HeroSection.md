# Module: `src/components/HeroSection.tsx`

## Overview

- **Purpose:** Renders the hero banner shown at the top of the home experience, including the headline, supporting copy, and jump-to-demo button.
- **Responsiveness:** Switches to a stacked layout on narrow viewports while keeping the CTA button prominent via `z-index` layering.

## Props

- `headline` — string for the marketing copy.
- `description` — supplemental paragraph text.
- `onDemoClick` — forwarded callback used to reveal the playable demo area.
